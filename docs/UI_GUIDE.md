# ConceptMaster UI Guide

## Product Feel

Use a bright, friendly learning-app style inspired by Duolingo, but keep CodeFair evidence accessible through menus rather than putting every card on the first screen.

## First Screen

- Brand: `오답 DNA 지도`.
- Subject switcher: active `수학 6학년`, locked `과학 준비중`.
- Menu: `학습`, `30초 데모`, `데이터`, `연구근거`.
- Hero: today's wrong-answer concept, misconception count, improvement progress, mascot, judge demo button.
- Learning path: five steps from wrong-answer record to review.

## Menu Responsibilities

- `학습`: primary student workflow and quiz.
- `30초 데모`: judge presentation flow.
- `데이터`: diagnosis cards, misconception map, logs, mastery, QA.
- `연구근거`: GitHub references, DTT trace, readiness audit.

## Responsive Rules

- No horizontal overflow on 390px mobile or 1220px desktop.
- Keep menu visible before dense evidence.
- Do not show raw internal IDs, fallback jargon, or English debug labels in the visible Korean UI.
