# 작품요약서 초안 - 심사기준 반영본

팀명: AI오답코치랩

작품명: 오답 DNA 지도(ConceptMaster)

참가자: 황건호

## 작품 요약

오답 DNA 지도(ConceptMaster)는 초등학교 6학년 학생이 수학 문제를 풀 때 같은 개념에서 반복해서 틀리는 문제를 해결하기 위한 AI 오답 코치 웹앱이다. 한국코드페어 SW공모전 주제인 "AI와 데이터로 해결하는 우리 사회의 문제"에 맞추어, 이 작품은 학습 중 반복 오답과 개념 이해 부족을 해결할 실질적 문제로 보았다. 학생이 정답과 해설만 보고 넘어가면 비슷한 문제에서 다시 틀릴 수 있기 때문에, 오답을 데이터로 남기고 같은 개념을 다시 확인하는 흐름이 필요하다고 판단했다.

작품은 `오답 기록 -> AI 진단 -> 같은 개념 재도전 -> 개선 확인` 순서로 작동한다. 학생이 문제를 풀면 `attempt_log`에 문제 ID, 개념 ID, 오개념 ID, 선택한 답, 정답 여부, AI 진단 출처, 다음 복습일이 저장된다. 오답이 발생하면 Manus API 또는 규칙 기반 fallback이 오답 원인을 설명하고, 같은 개념의 재도전 문제를 추천하거나 생성한다. 재도전 결과는 `concept_summary`에 반영되어 개념별 오답률, 대표 오개념, 재도전 성공 여부, 오늘 복습할 개념으로 정리된다.

기존 문제풀이 앱은 정답과 해설을 보여준 뒤 다음 문제로 넘어가는 경우가 많다. 이 작품은 그 방식과 달리, 오답을 하나의 학습 데이터로 저장하고 AI 진단과 재도전 결과까지 연결한다. 그래서 학생은 "몇 문제를 틀렸는지"가 아니라 "어떤 개념을 왜 반복해서 틀렸고, 다시 풀었을 때 나아졌는지"를 볼 수 있다. 이것이 이 작품의 차별점이자 창의성이다.

현재 구현은 초6 수학 대표문항 6개를 중심으로 심사용 데모를 구성했다. 데모를 실행하면 일부러 오답을 기록하고, AI 또는 fallback 진단을 보여준 뒤, 같은 개념 재도전 문제를 풀게 하고, 마지막에는 오답률, 개선률, `attempt_log`, `concept_summary`를 화면에 보여준다. 최신 브라우저 검증에서는 오답 후 재도전 정답을 맞히면 이전 오답이 `회복`으로 바뀌고 개선률이 50%로 갱신되는 것을 확인했다. 자동 테스트에서는 문제 은행 검증, 학습 기록 저장, 같은 개념 재도전, 대시보드 계산, AI 실패 시 fallback 처리, API 키 비노출, 30초 데모 계약, 15분 인수인계 계약, M6 인간 검증 게이트 등 44개 테스트가 통과했다.

AI 결과는 무조건 정답으로 처리하지 않는다. 생성 문항은 사람 검수 전까지 `needs_review` 상태로 표시하고, Manus API가 느리거나 실패해도 fallback으로 데모가 멈추지 않도록 했다. 앞으로는 실제 학생 풀이 데이터를 더 모아 오개념 분류를 보완하고, 초6 수학 단원별 문제 수를 늘리며, 교사용 검수 화면을 추가할 계획이다. 이 작품은 AI가 답을 대신 알려주는 앱이 아니라, 오답 데이터를 이용해 학생이 자신의 약한 개념을 찾고 다시 이해했는지 확인하도록 돕는 AI 학습 회복 도구이다.
